== Description ==

Provides the ability to quickly post a blog item to your facebook mini feed. This is done by adding a button to
the add/edit forms for posts and pages.

== Installation == 

1. Extract the contents of post-to-facebook.zip and upload to the wp-content/plugins folder.
2. Activate the plugin through the 'Plugins' menu in WordPress
